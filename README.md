<h1 align="center">2nd Chance 4 Pets Repository</h1>

### Running the backend

Navigate to the backend folder

```
sc4p-Backend
```

Install dependencies

```
npm i
```

Start server (port 3001)

```
npm start
```

### Running the frontend (React)

Navigate to the frontend directory

```
sc4p-Frontend
```

Install dependencies

```
npm i
```

Start app

```
npm start
```
